💈 Black Gold Barbearia — Site Institucional
Site institucional moderno e responsivo para barbearia, desenvolvido com HTML5 e CSS3 puro.

📋 Sobre o Projeto
Site de uma página (one page) criado para apresentar os serviços, localização e facilitar o agendamento de clientes via WhatsApp. O design segue uma identidade visual premium com tons de preto e dourado.

🖥️ Tecnologias Utilizadas

HTML5
CSS3 (variáveis, grid, flexbox, animações)
JavaScript vanilla (animações de scroll)
Google Fonts (Playfair Display + Barlow)


📁 Estrutura de Arquivos
barbearia-blackgold/
├── index.html      # Estrutura principal do site
├── style.css       # Estilos separados
└── README.md       # Este arquivo

📌 Seções do Site

Hero — Título principal com chamada para ação
Sobre — História e diferenciais da barbearia
Serviços — Cards com preços e duração
Galeria — Fotos dos trabalhos realizados
Depoimentos — Avaliações de clientes
Localização — Endereço, horários e mapa
Contato — Botões de agendamento e redes sociais
